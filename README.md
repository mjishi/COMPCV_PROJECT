# COMPCV_PROJECT

Project for computer vision fall 2025. this project detects intruders using MOG2 and draws a bounding rectangle on them.

# Setup
Make sure to have the following packages installed

Dependencies:
opencv
numpy

your computer needs to have a camera built-in or be connected to a webcam.

# Instructions

1. Download notebook file
2. Run all cells
3. to exit the program, click Q on your keyboard

# Troubleshooting

if camera isnt detected change the number in
cap = cv2.VideoCapture(0)
from 0 to 1,2, etc. until camera is shown
